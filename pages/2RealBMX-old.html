2Real BMX

Summary of Project

2-Real BMX is a BMX simulation game which uses semi-realistic physics. It turns out that after figuring out how to make a bicycle steer the proper amount at the right time to ensure stability that the physics model created was more suitable for a motorcycle than a bike.

How it works

The bike uses Unity 5's built in wheel colliders and rigidbody in addition to a physics wrapper which enables the unity wheel colliders to be used with two wheels. By default the built in colliders do not support two wheeled vehicles. A faux gyroscopic force is applied to the bikes rigidbody to maintain stability while at rest and while moving. Depending on the speed of the bike more or less force is applied. When the bike accelerates the handle bars are progressively turned less and the bike begins to lean farther. 

// Steering amount based on speed.
float gyro = Mathf.Clamp01((relativeVelocity.z) / 20f);
float gyroSteer = (1 - gyro) * 0.9f;
steeringLean = steeringAngle * gyro;
barTurnAmount = steeringAngle - steeringLean * gyroSteer;

rigidbody.AddRelativeTorque(0, 0, -steeringLean / 2f);
bike.SteeringPoint.localRotation = defaultAngle; // Don't allow continuous spin
TurnFrontFork(barTurnAmount);

The BMX bike can perform a few tricks including spinning and flipping as well as a bar spin. Implementing the ability to grind was partially completed using a magnetic force to attract the bike to the rail when the pegs are within a certain distance to the rail. All actions are based on the bikes current state which is determined by how many wheels are currently grounded. After all, It would not make sense to allow the bike to backflip while grounded without first performing at least a bunny hop.

The bike is controlled using an Xbox controller for which an input handler was creating thereby simplifying the task of accessing button and axis events in code. Keyboard support was also added at the request of play testers as not all play testers had access to an Xbox controller. Using Unity5's input event system the ability to navigate menu its was added simply by setting an initial focus point which was then manipulated using the controllers left joystick.

All of the ramp models were created by myself using 3ds Max. The palm tree is a unity standard asset and can be found on the asset store by searching "palm tree". The bike model can be found at the link provided in the resources section below.
